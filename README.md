**# Wheeldeal
WheelDeal is a digital marketplace designed to simplify the buying and selling of pre-owned cars. It enables users to discover trusted second-hand vehicles at competitive prices through a secure and seamless platform. WheelDeal also empowers car owners to sell their vehicles directly to verified and authenticated buyers, ensuring transparency, reliability, and a smooth transaction experience for all users.
**
## Features
- User registration and login
- Vehicle listing and details
- Search functionality
- Admin management
- Responsive UI using Bootstrap

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
